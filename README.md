# 索尼风味光学变焦 | Sony Style Optical Zoom

一个为 Minecraft 1.21.1 Fabric 打造的索尼风格光学变焦模组，还原索尼摄影机的 W-T 变焦体验。

A Sony-style optical zoom mod for Minecraft 1.21.1 Fabric, recreating the iconic W-T zoom experience from Sony camcorders.

---

## 功能特性

- **平滑光学变焦** — 按住按键进入变焦模式，滚轮调节 1-50 倍
- **W-T 变焦条** — 左上角显示专业摄影机风格的变焦指示条
- **变焦锁定** — 切换锁定模式，解放按键
- **预设跳转** — 三个可配置预设，一键跳到常用倍率
- **截图功能** — 带闪光反馈的截图功能
- **指令配置** — 游戏内指令动态调整预设和最大倍率

## Features

- **Smooth Optical Zoom** — Hold key to zoom, scroll wheel to adjust 1-50x
- **W-T Zoom Bar** — Professional camcorder-style zoom indicator in top-left
- **Zoom Lock** — Toggle lock mode to free up your key
- **Preset Jump** — Three configurable presets for quick zoom levels
- **Screenshot** — Capture screenshots with flash feedback
- **Command Config** — Adjust presets and max zoom in-game

## 按键说明 / Key Bindings

| 按键 / Key | 功能 / Function |
|------|------|
| `LCtrl` | 按住变焦 / Hold to zoom |
| `N` | 切换变焦锁定 / Toggle zoom lock |
| `Numpad 1` | 预设 1 / Preset 1 |
| `Numpad 2` | 预设 2 / Preset 2 |
| `Numpad 3` | 预设 3 / Preset 3 |
| `F2` | 截图 / Screenshot |
| `]` | 增大最大倍率 (+10x) / Increase max zoom |
| `[` | 减小最大倍率 (-10x) / Decrease max zoom |
| `滚轮 / Scroll` | 变焦时调节倍率 / Adjust zoom level |

## 指令用法 / Commands

| 指令 / Command | 功能 / Function |
|------|------|
| `/zoom preset <1\|2\|3> <value>` | 设置预设倍率 / Set preset zoom (1.0-200.0) |
| `/zoom maxzoom <value>` | 设置最大倍率 / Set max zoom (10.0-200.0) |
| `/zoom reset` | 重置所有设置 / Reset all settings |
| `/zoom info` | 查看当前配置 / Show current config |

## 安装方法 / Installation

1. 安装 [Fabric Loader](https://fabricmc.net/use/installer/) / Install [Fabric Loader](https://fabricmc.net/use/installer/)
2. 下载 [Fabric API](https://modrinth.com/mod/fabric-api) / Download [Fabric API](https://modrinth.com/mod/fabric-api)
3. 将本模组的 `.jar` 文件放入 `mods` 文件夹 / Put the `.jar` file into `mods` folder
4. 启动游戏 / Launch the game
